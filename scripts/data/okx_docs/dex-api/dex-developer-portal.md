开发者平台
#
为了帮助你更高效地构建自己的应用，本篇将介绍开发者管理平台的功能以及最佳使用方法。
注册/登陆
#
点击导航栏
开发者中心
下的
开发者管理平台
。
点击
连接钱包
创建或登录账户。我们推荐选择 OKX Web3 钱包 (无缝并快速) ，你也可以使用其他钱包。
选择好钱包后，点击
验证
，钱包处将弹出一个窗口。在钱包内
确认签名
。
恭喜！你已成功创建账户，并登录至开发者管理平台，为方便您的使用，我们已创建好一个默认项目供您浏览。
生成新的 API key
#
在需要创建 API key 的项目处点击 “API key” 按钮。
点击创建 API key 按钮。
输入 API key 的名称和密码(Passphrase)，点击创建生成相应的 API key。
一旦密码(Passphrase) 丢失，将无法再使用 API key。请牢记密码(Passphrase)。
查看 secret key
#
在向 API 发起请求时，需要也需要提供 Secret key。点击右侧的复制按钮，可查看你的 secret key。
预备知识
#
API key：为应用程序或用户提供一个唯一标识，使其能够与 API 进行交互。
Secret key：与 API key 配合使用，通过增加一道安全令牌来提供额外的安全保证。
密码(Passphrase)： 用于在服务端加密 secret key，提供鉴权和安全防范。查看 Secret key、修改 API 信息也会用到密码(Passphrase)。
