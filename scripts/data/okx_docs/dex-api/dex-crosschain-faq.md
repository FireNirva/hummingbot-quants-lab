FAQ
#
常见的跨链失败原因：
#
跨链交易失败分为源链和目标链兑换失败以及跨链桥交易失败。
源链和目标链兑换交易失败，可能由以下原因：：
广播失败
Gas 不足
超出滑点设置
币种授权额度不足
第三方 DEX 合约执行报错
交易被加速或取消
交易的代币为 safemoon 类代币或者貔貅币
跨链桥交易失败，可能由以下原因：
超出三方桥的滑点
触发三方桥黑名单
未满足跨链桥最小/最大跨链数量限制
跨链桥流动性不足
跨链桥交易超时
交易时间过长：
#
源链兑换交易时间一般耗时在一分钟内，官方跨链桥因为自身跨链机制问题交易时间较长，其余三方跨链桥如果长时间未返回交易终态，可以查询链上交易状态或者联系三方桥客服进行确认。
是否支持加速或取消交易：
#
不支持加速或取消。
跨链退款：
#
跨链退款目前分为两种类型，第一种是第三方跨链桥发起的退款，第二种则是欧易 DEX 发起的退款。
第三方跨链桥发起的退款
退款的主要原因是资金进入跨链桥后触发跨链桥退款规则未能正常进行跨链桥交易。
常见的退款原因一般为跨链桥池子变动导致流动性不足、用户跨链金额未满足跨链桥最小跨链数量，或者触发第三方桥黑名单等其他规则。
退款交易由第三方跨链桥发起，退款的币种一般为 USDT、USDC、DAI、WETH、ETH，退款的币种数量由第三方跨链桥返回，用户最终获得的退款数量为三方桥扣除完链上网络费后的数量。
常见的跨链退款的三方桥有 SWFT 等。
欧易 DEX 发起的退款
退款的主要原因有两个，第一个是在进行目标链兑换时欧易会提前预执行模拟交易该笔兑换交易，如果预执行交易报错则会触发自动退款。第二个是目标链交易链上失败也会触发自动退款。
预执行模拟交易结果报错的常见原因一般为超出滑点设置、第三方 DEX 合约执行报错，或者交易的代币为 safemoon 类代币或者貔貅币等。
目标链交易链上失败触发自动退款，链上失败的原因可以参考上面的常见的跨链失败原因。
退款交易由欧易 DEX 发起，退款的币种为跨链桥跨出的币种，一般是 USDT、USDC、DAI、WETH、ETH。退款的币种数量由欧易 DEX 返回，用户最终获得的退款数量为扣除完链上网络费后的数量。
