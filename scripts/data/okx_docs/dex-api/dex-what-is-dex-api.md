什么是 DEX API
#
介绍
#
欢迎来到 OKX DEX 开发者文档。
OKX DEX 是一站式多链跨链聚合交易平台，目前支持包括 EVM、Solana、SUI 等在内的 20 多条网络，致力于提供最实时、最优的交易报价。
OKX DEX API 提供了一套强大的核心功能组合，涵盖
单链现货交易、兑换指令执行、跨链兑换、交易模拟和上链
以及
实时行情与链上数据分析
等功能，完整覆盖去中心化交易系统的核心模块。
通过标准化的交易服务接口，能够帮助开发者简化开发流程，大大缩短开发时间和成本。
你可通过本文档构建适用于
兑换应用
、
跨链桥方案
行情数据网站
、
Web3钱包
交易机器人
等多样化场景的 Web3 交易服务与应用。对于希望自建 Web3 交易基础设施的企业或开发者而言，采用 OKX DEX API 是更高效且更具成本效益的解决方案。
